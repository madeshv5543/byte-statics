# Obyte statistics
A collection of scripts to render various stats about the Obyte network

## Requirements
* Php 5+
* SQLite 3+
* Perl 5+
* Obyte full node
* jquery-2.2.4.min.js
* ipstack.com API key. 
* A licence for HighStock js from https://www.highcharts.com (free for non commercial purpose)
